# D7. 画面UIワイヤーフレーム

## 1. 共通ルール

### 1.1 内部解像度
- **1280 × 720** 固定(16:9)
- ブラウザウィンドウにフィットするよう**等比スケール**(letterbox)
- すべての座標・サイズは内部解像度基準

### 1.2 座標系とセーフエリア
- 原点: 左上 `(0, 0)`、右下 `(1280, 720)`
- セーフエリア: 画面四辺から **24px** 内側を基準

### 1.3 タイポグラフィ
| ロール | サイズ | フォント |
|---|---|---|
| H1(タイトルロゴ) | 120px | ゲーム用ロゴフォント |
| H2(見出し) | 40px | 丸ゴシック系 Bold |
| H3(小見出し) | 28px | 丸ゴシック系 Bold |
| Body | 18px | 丸ゴシック系 Regular |
| Caption | 14px | 丸ゴシック系 Regular |
| Score | 36px | Digital font(等幅) |
| Chain Text | 48〜96px | ロゴ類似、アウトライン強め |

### 1.4 カラーパレット
| 役割 | 色 |
|---|---|
| 背景ベース | `#1a1a2e`(夜空系)|
| アクセント(既定) | `#ffd60a`(黄) |
| 成功/勝利 | `#52ff8f` |
| 警告/ピンチ | `#ff5c5c` |
| Info | `#5cb8ff` |
| テキスト | `#ffffff` / `#c8c8d8`(弱) |
| パネル | `rgba(20, 20, 40, 0.85)` |

> キャラクター選択時は、UI アクセント色が**選択キャラのイメージカラー**に変化。

## 8. シーン 6: Match

### 8.1 ソロ(1人)レイアウト
```
┌───────────────────────────────────────────────────────────────┐
│  ⏸ Pause(Esc)        SOLO ENDLESS            🎨4 colors       │
│                                                               │
│  ┌───────────┐              ┌───────────────┐  ┌──────┐       │
│  │           │              │               │  │ NEXT │       │
│  │ Character │              │  Main Field   │  │ ┌──┐ │       │
│  │  Portrait │              │   6 × 12      │  │ │N1│ │       │
│  │           │              │               │  │ └──┘ │       │
│  │           │              │               │  │ ┌──┐ │       │
│  │           │              │               │  │ │N2│ │       │
│  │           │              │               │  │ └──┘ │       │
│  │           │              │               │  └──────┘       │
│  │           │              │               │   SCORE         │
│  └───────────┘              └───────────────┘   12340         │
│  Kiria                                         CHAIN 3 / 7    │
└───────────────────────────────────────────────────────────────┘
```

**寸法**:
- Main Field: `40px × 6 = 240px 幅`、`40px × 12 = 480px 高`
- Character Portrait: 240 × 360

### 8.2 対戦2人レイアウト
- Main Field 左右対称、各 `40px × 6 = 240` 幅
- 中央に NEXT(自分の分)表示
- フィールド上部に予告ぷよバー

### 8.3 対戦3人レイアウト
自分のフィールドを左寄せ大きく、相手 2 人を右側に 2段積み小さく表示。
- 自分(左): `40px × 6 = 240` 幅、`40px × 12 = 480` 高
- 相手(右上段・右下段): セル `16px`、`16×6=96` 幅 × `16×12=192` 高

### 8.4 対戦4人レイアウト
- 自分: 右側に大きく(240×480)
- 相手 3 人: 左側に小さく並べる

### 8.5 共通要素

#### 予告ぷよバー
```
[🌟][🌙][🪨][🪨][🫧][🫧]
  大きいアイコン順に左詰め、最大 6 個、余りは右端 "+" 表示
```

#### 連鎖テキスト(演出)
- フィールド中央に **"3 れんさ!"** と大きく表示
- 連鎖数に応じてサイズ増加、色がキャラの image color
- 20 フレーム表示、フェードアウト

#### 敗北演出
- 敗北プレイヤーのフィールドに **"BAD END" オーバーレイ**
- 60 フレーム演出後、フィールドがグレースケールに

#### 観戦中の自フィールド
- 半透明(α=0.3)+ "SPECTATING" ラベル

## 14. フォーカス管理とナビゲーション

| キー | 共通アクション |
|---|---|
| `Enter` / `Space` | 決定 |
| `Esc` | 戻る / キャンセル |
| `Tab` | 次の選択要素(DOM フォーカス) |
| `↑↓←→` | グリッド/リスト内移動 |

- すべてのボタンはクリック可能
- フォーカスリング視認性(アクセント色で 2px)
- モーダル表示中はフォーカストラップ

## 15. トランジション

| from → to | 演出 |
|---|---|
| Title → ModeSelect | スライドイン(左) |
| ModeSelect → CharacterSelect | フェード |
| ロビー → Room | フェード + パネル拡大 |
| Room → Match | カウントダウン "3,2,1" で画面遷移 |
| Match → Result | フィールドが中央に縮んで、Result テーブルが展開 |
| 任意 → Settings | フェードイン(半透明背景)|

持続時間は一律 **200〜300ms**。

---

## 16. アクセシビリティ

- 色だけで情報を伝えない(READY 状態はアイコンも併用)
- テキストコントラスト WCAG AA 相当
- `aria-label` を主要要素に付与
- キー操作のみで**全メニュー到達可能**
- 文字サイズ: 最小 14px 以上

## 17. レスポンシブ・モバイル

### 17.1 初期版の方針
- **PC ブラウザのみ対応**
- ウィンドウサイズに応じ 16:9 で中央配置、letterbox
- 最小ウィンドウ 960×540 未満では警告表示

### 17.2 将来対応(範囲外)
- タッチ操作、縦長レイアウト
